编写一个程序，生成以下输出：
Brazil, Russia, India, China
India, China,
Brazil, Russia
除了 main()   以外，该程序还要调用两个自定义函数：一个名为
br()
   ，调用一次打印一次“Brazil,    Russia   
”；另一个名为 
ic()
   ，调用一次打印一次“India,    China   ”。
其他内容在
main()   函数中完成。

同4，自定义函数，并定义函数
br()
和
ic()
br	输出
Brazil,    Russia   ”；
ic() 输出： 
India,    China   ”


代码： 
#include<stdio.h>

void br(void);
void ic(void);
int main(void) {
	printf("Brazil, Russia, India, china\n");
	ic();
	printf(",\n");
	br();

	return 0;
}

//定义函数值
void br(void) {
	printf("Brazil, Russia\n");
}
void ic(void) {
	printf("India, China");
}
