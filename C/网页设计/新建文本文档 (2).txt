要将你的文档章节和小结做成一个HTML文档，并实现目录链接，可以使用HTML的锚链接功能。下面是一个示例代码，展示了如何将文档分章节并添加一个目录，目录中的链接可以直接跳转到对应的章节。

```html
<!DOCTYPE html>
<html lang="zh">
<head>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>文档目录</title>
 <style>
 body {
 font-family: Arial, sans-serif;
 line-height: 1.6;
 margin: 20px;
 }
 h2, h3 {
 color: #333;
 }
 .toc {
 margin-bottom: 20px;
 }
 .toc a {
 text-decoration: none;
 color: blue;
 }
 .toc a:hover {
 text-decoration: underline;
 }
 </style>
</head>
<body>

 <h1>文档标题</h1>

 <div class="toc">
 <h2>目录</h2>
 <ul>
 <li><a href="#章1">1 章</a></li>
 <li><a href="#章1.1">1.1 小结</a></li>
 <li><a href="#章1.2">1.2 小结</a></li>
 <li><a href="#章1.3">1.3 小结</a></li>
 <li><a href="#章2">2 章</a></li>
 <li><a href="#章2.1">2.1 小结</a></li>
 <li><a href="#章2.2">2.2 小结</a></li>
 <li><a href="#章2.3">2.3 小结</a></li>
 <li><a href="#章3">3 章</a></li>
 <li><a href="#章3.1">3.1 小结</a></li>
 <li><a href="#章3.2">3.2 小结</a></li>
 <li><a href="#章3.3">3.3 小结</a></li>
 </ul>
 </div>

 <h2 id="章1">1 章</h2>
 <p>这里是第一章的内容...</p>

 <h3 id="章1.1">1.1 小结</h3>
 <p>这里是第一章第一小结的内容...</p>

 <h3 id="章1.2">1.2 小结</h3>
 <p>这里是第一章第二小结的内容...</p>

 <h3 id="章1.3">1.3 小结</h3>
 <p>这里是第一章第三小结的内容...</p>

 <h2 id="章2">2 章</h2>
 <p>这里是第二章的内容...</p>

 <h3 id="章2.1">2.1 小结</h3>
 <p>这里是第二章第一小结的内容...</p>

 <h3 id="章2.2">2.2 小结</h3>
 <p>这里是第二章第二小结的内容...</p>

 <h3 id="章2.3">2.3 小结</h3>
 <p>这里是第二章第三小结的内容...</p>

 <h2 id="章3">3 章</h2>
 <p>这里是第三章的内容...</p>

 <h3 id="章3.1">3.1 小结</h3>
 <p>这里是第三章第一小结的内容...</p>

 <h3 id="章3.2">3.2 小结</h3>
 <p>这里是第三章第二小结的内容...</p>

 <h3 id="章3.3">3.3 小结</h3>
 <p>这里是第三章第三小结的内容...</p>

</body>
</html>
```

### 说明：

1. **目录部分**：
 - 使用了一个无序列表 `<ul>`，每个列表项 `<li>` 内含一个链接 `<a>`。链接的 `href` 属性指向相应的小节（例如 `#章1`，`#章1.1` 等），这样点击后会跳转到指定位置。

2. **章节及小结部分**：
 - 每个章节和小结的标题都使用了 `id` 属性，这个 `id` 对应目录中链接的 `href` 值。

3. **样式设置**：
 - 使用了一些基本的CSS样式，使文档看起来更美观。

你可以将这段代码复制到一个 `.html` 文件中，然后用浏览器打开查看效果。如果有其他具体需求，可以进一步调整。
