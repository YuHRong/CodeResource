# CSS基础复习

## CSS基础

### 什么是CSS

统一样式表（`CSS`）是一种标记语言，用于在`HTML`元素上应用样式。CSS用于`颜色`、`背景图像`、`布局`等。

### CSS 规则的基本剖析

CSS 规则由两个主要部分组成：`选择器`和`声明块`。选择器是 `CSS` 中使用的`一种模式`，用于`识别`和`针对特定`的 `HTML` 元素设计样式。`声明块`为给定的一个或多个`选择器应用一组样式`。

这是一个 `CSS` 规则的通用语法： 

selector `{`

property`:` value`;`


}


### meta name`=`"viewport"元素

此 `meta` 元素为浏览器提供了如何在不同设备（尤其是`手机`和`平板电脑`）上控制`页面尺寸`和`缩放比例`的说明。

### 默认浏览器样式

每个 `HTML` 元素都将应用默认浏览器样式。这通常包括`默认外边距`和`内边距`等项目。

## 内联、内部和外部 CSS

### 内联 CSS

这些样式可以使用 `style` 属性直接写入 `HTML` 元素中。出于分离的考虑，大多数情况下你不会使用`内联 CSS`。

这是一个内联 `CSS` 的示例

 <`link` rel`=`"`stylesheet`" `href`=`"`styles`.`css`"`>


<`p` style`=`"`color`: `red`;`"`>`This is a red paragraph.`<`/p`>


### 内部 CSS

这些样式写在 `HTML` 文档 `head`部分的 `<`style`>` 标签内。这对于创建`简短的代码示例`很有用，但通常不需要使用内部 `CSS`。

### 外部CSS

这些样式被写入一个单独的`CSS文件`，并使用 `head`部分的 `link` 元素链接到`HTML`文档。在大多数项目中，你都会使用`外部CSS文件`，而不是`内部`或`内联CSS`。

## 使用 `width`和 `height`属性

### width属性

该属性指定元素的`宽度`。如果未指定宽度，则默认设置为 `auto` 。这让浏览器根据元素的`内容`、`父元素`和`显示类型`来确定元素的宽度。

### min-width属性

该属性指定元素的最小宽度。

### max-width属性

该属性指定元素的最大宽度。

### height属性

该属性指定元素的高度。相同，高度默认为 `auto` ，这意味着它会根据`内部内容`进行调整。

### min-height属性

该属性指定元素的最小高度。

### max-height属性

该属性指定元素的最大高度。

## 不同类型的 CSS 组合选择器

### 后代选择器

该选择器用于选择指定父系元素的`后代`元素。下面的示例将选择 `ul` 元素内的所有 `li`项。 

<`link` rel`=`"`stylesheet`" `href`=`"`styles`.`css`"`>


<`ul`>

<`li`>`Example item one`<`/li`>

<`li`>`Example item two`<`/li`>

<`li`>`Example item three`<`/li`>


<`/ul`>

ul `li` {

background-color`:` yellow`;`


}


### 子代组合器（ `>`） 

此组合器用于选择`指定父元素`的`直接子代元素`。以下示例将定位所有作为 `container`类`直接子代`的 `p元素`。

<`link` rel`=`"`stylesheet`"` href`=`"`styles`.`css`"`>


<`div` class`=`"`container`"`>`

<`p`>`This will get styled.`<`/p`>



<`div`>

<`p`>`This will not get styled.`<`/p`>

<`/div`>


<`/div`>

 .container > p `{`

background-color`:` black`;`

color`:` white`;`


}


### 相邻兄弟选择器（ 选择 `+`）

该选择器紧跟`指定同级`元素的元素。 下面的示例将选择紧随 `h2`元素后面的段落元素。 

<`link` rel`=`"`stylesheet`" `href`=`"`styles`.`css`"`>


<`h2`>`I am a sub heading`<`/h2`>




<`p`>`This paragraph element will get a red background.`<`/p`>

h2 + p `{`

background-color`:` red`;`


}


### 后续兄弟选择器（ `~`） 

该选择器选择指定元素后面的`所有同级`元素。下面的示例将只对`第二个段落`元素进行`样式设置`，因为只有它是 `ul` 元素的`同级`元素，并且`共享相同`的父元素。 

<`link` rel`=`"`stylesheet`" `href`=`"`styles`.`css`"`>


<`div` class`=`"`container`"`>`

<`p`>`This will not get styled.`<`/p`>

<`ul`>

<`li`>`Example item one`<`/li`>

<`li`>`Example item two`<`/li`>

<`li`>`Example item three`<`/li`>

<`/ul`>

<`p`>`This will get styled.`<`/p`>


<`/div`>


<`p`>`This will not get styled.`<`/p`>

 ul ~ p `{
`
background-color`:` green`;`


}


## 内联、块和内联块等级元素

### 内联元素

内联元素只占用所需的`宽度`，并且不会从新行开始。这些元素在内容中流动，允许`文本`和`其他内联`元素出现在它们旁边。常见的`内联`元素有 `span`、 `anchor`和 `img`元素。

### 块级元素

块级元素默认从`新行`开始，引发容器的`全部可用宽度`，水槽容器的宽度。一些常见的`块级元素`有 `div`、 `paragraph`和 `section`元素。

### 内联块等级元素

你可以使用 `display`属性将元素设置为 `inline-block` 。这些元素的行为类似于`内联`元素，但可以像`块等级`元素一样设置 `width`和 `height`。

## 外边距和内边距

### margin属性

该属性用于在`元素外部`、`相似元素`和`周围元素`之间设置空间。

### padding属性

该属性用于在元素内部的`内容`和`未知`之间设置空间。

### margin简单写

你可以指定 `1`-`4` 个值来设置各个`外边距`。一个值适`用于所有`四个边；`两个值设置`的先是 `top`和 `bottom`，然后是 `right`和 `left`；三个值设置的先是 `top`、`水平的边`（ `right`和 `left`），然后是 `bottom`；四个值依次设置是 `top`、 `right`、 `bottom`、 `left`。

### padding简单写

你可以指定 `1`-`4` 个值来设置`各个内边距`。一个值适用于`所有四个边；`两个值设置的先是 `top`和 `bottom`，然后是 `right`和 `left`；三个值设置的先是 `top`、`水平的边`（ `right`和 `left`），然后是 `bottom`；四个值依次设置是 `top`、 `right`、 `bottom`、 `left`。

## CSS 优先级

### 内联 `CSS` 优先级

内联 `CSS` 的优先级最高，因为它直接`评估元素`。它会覆盖`任何内部`或`外部 CSS`。内联样式的优先级值为 `(`1`,` 0`,` 0`,` 0`)`。

### 内部`CSS`优先级

内部`CSS`是在`HTML`文档 `head`部分的 `style` 元素中定义的。它比`内联样式`的优先级低，但可以覆盖`外部样式`。

### 外部`CSS`优先级

外部`CSS`通过 `head`部分的 `link` 元素进行链接，并写入单独的 `.css`文件中。它的优先级`最低`，导致大型项目提供最佳的可维护性。

### 通用选择器（ `*`）

是一种特殊的选择 `CSS 器`，可匹配`文档`中的任何元素。它通常用于将`一种样式`评估页面上的所有元素，这对于`不同浏览器`中重置或规范样式非常有用。在所有`选择器`中，通用选择器的`优先级值`最低。它对`优先级值`的所有部分 `(`0`,` 0`,` 0`,` 0`)` 的贡献为 `0`。

### 类型选择器

这些选择器根据`元素的标签名称`选择元素。与其他选择器相比，`类型选择器`的优先级相对`较低`。类型选择器的`优先级值`为`(`0`,` 0`,` 0`,` 1`)`。

### 类选择器

这些选择器由一个句点（ `.` ）后类命名确定。`类选择器`的优先级值为`(`0`,` 0`,` 1`,` 0`)`。这意味着`类选择器`可以覆盖`类型选择器`，但它们可以跟`ID选择器`和`内联样式`覆盖。

### ID 选择器 `#`

ID 选择器由哈希符号 `(` ) `后跟` ID 名称`定义`。  `ID选择器`的优先级选择器非常高，比`类型器`和`类选择器`低，但低于`内联样式`。ID选择器的优先级值为`(`0`,` 1`,` 0`,` 0`)`。

### !important关键字

用于赋予样式规则`最高优先级`，设置优先于某个`属性`的`声明`。使用时，它会强制浏览器应用指定的`样式`，而不考虑其他`选择器`的优先级。使用 `!important`时应其他规则，因为它使 `CSS` 更难以维护和调试。

### 级联算法

当有多种样式针对相同元素时，用于决定应用哪些 `CSS` 规则的算法。它确保根据一组定义的明确的规则，使用`最合适`的样式。

### CSS继承样式

从`父元素`向`子元素`传递的过程。通过`继承`，可以在`文档树`的上层定义`样式`，放入其所需的多个元素，而无需为每个元素明确指定样式。
