## 了解结构

结构是 C# 中的`值类型`，这意味着它们直接存储其数据，而不是引用数据。 使用`struct`关键字定义结构。 结构通常用于以数据为中心的小型类型，这些类型提供最少的行为。
- 结构具有值语义，这意味着在赋值或方法调用期间复制其值。
- 常见用例包括表示数字、布尔值、字符和时间实例。
- 结构通常用于表示轻型数据，如几何形状（例如，、`Rectangle`<b>Circle<b>）、坐标或配置设置。

结构最适合小型、不可变和性能关键型数据。 对于较大的、更复杂的对象或行为是焦点时，请考虑改用类。 与类不同，结构不能继承自其他结构或类，但它们可以实现接口。
*提示*如果类型侧重于行为而不是数据，则类通常是一个更好的选择，因为类的引用语义。

## 定义和实现结构

```c#
public struct Point
{
    public int X;
    public int Y;

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }
}
```

代码示例演示如何使用 C# 中的字段和构造函数定义结构。

结构还可以实现接口，但不能从其他类或结构继承。

## 使用结构封装数据

封装允许你控制对结构中的数据的访问。 使用访问修饰符（例如`private`、`public`或`internal`指定辅助功能。
- 字段和属性可以设置为专用，以限制直接访问。
- 方法可以提供与数据交互的受控方法。<br>例如：
```c#
public struct Rectangle
{
    private int width;
    private int height;

    public Rectangle(int width, int height)
    {
        this.width = width;
        this.height = height;
    }

    public int Area => width * height;
}
```

代码示例演示 C# 中的结构，其中包含封装的字段和区域的计算属性。

## 将只读修饰符应用于结构，以便实现不可变性

不可变性可确保在创建结构后，无法更改其数据。 在处理整个生存期内应保持不变的数据（例如坐标或配置值）时，结构非常有用。 将`readonly`修饰符应用于结构时，可以强制实施不可变性，使代码更具可预测性，并且更容易出现意外修改导致的错误。

```c#
public readonly struct ImmutablePoint
{
    public int X { get; init; }
    public int Y { get; init; }

    public ImmutablePoint(int x, int y)
    {
        X = x;
        Y = y;
    }
}
```

代码示例演示如何使用 readonly   修饰符在 C# 中创建不可变结构。*注*虽然结构本身是不可变的，但其中的可变引用类型仍可以更改其状态。

## 将对象初始值设定项与结构配合使用 分组

使用结构时，通常需要初始化多个属性。 对象初始值设定项提供了一种简单且可读的方式，用于在单个步骤中创建和初始化结构实例。 此方法可减少重复代码，并明确意图。

下面的代码示例定义了一个结构，该结构包含将相关数据组合在一起的属性，并使其更易于初始化和使用结构实例。

```c#
public struct Point
{
    public int X { get; set; }
    public int Y { get; set; }
}
```

使用对象初始值设定项，可以创建和初始化 ：`Point`

```c#
var point = new Point { X = 10, Y = 20 };
```

代码示例演示如何在创建结构实例时使用对象初始化器设置属性值。*提示*使用具有多个属性的结构时，对象初始值设定项使代码更加简洁、更易于阅读。

## 实际示例：使用结构表示二维空间中的点

结构通常用于表示小型不可变的数据结构。 例如，可以使用结构来表示 2D 空间中的点，并计算两个点之间的距离：

```c#
public struct Point
{
    public int X { get; }
    public int Y { get; }

    public Point(int x, int y)
    {
        X = x;
        Y = y;
    }

    public double DistanceTo(Point other)
    {
        int dx = X - other.X;
        int dy = Y - other.Y;
        return Math.Sqrt(dx * dx + dy * dy);
    }
}

var point1 = new Point(3, 4);
var point2 = new Point(7, 1);
Console.WriteLine($"Distance between points: {point1.DistanceTo(point2)}");
```

此示例演示如何为某个点定义结构、计算两个点之间的距离，以及使用简单程序中的结构。

结合封装、不可变性和对象初始值设定项，可以创建高效且可预测的结构。 这些功能可帮助你以干净有序的方式管理相关数据。
